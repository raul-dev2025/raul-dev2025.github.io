Preparación para el Examen - Módulo 0852

Instalación y actualización de sistemas operativos

Temas Clave

1. Arquitecturas de un Sistema Microinformático

  • Subsistemas:

    • Hardware: CPU, memoria, dispositivos E/S.

    • Software: Sistema operativo, aplicaciones.

  • CPU:

    • Componentes: ALU, Unidad de Control, registros.

  • Memoria:

    • Tipos: RAM (volátil), ROM (no volátil), caché.

  • Dispositivos:

    • Almacenamiento: HDD, SSD, USB.

    • E/S: Teclado, ratón, monitor.

  • Buses:

    • Tipos: Datos, dirección, control.

    • Características: Ancho de banda, velocidad.

2. Funciones del Sistema Operativo

  • Conceptos básicos:

    • Procesos: Programas en ejecución.

    • Archivos: Unidades lógicas de almacenamiento.

    • Kernel: Núcleo del SO.

  • Funciones principales:

    • Interfaz de usuario (CLI o GUI).

    • Gestión de recursos (CPU, memoria).

    • Administración de archivos y tareas.

3. Elementos de un Sistema Operativo

  • Gestión de procesos y memoria.

  • Sistemas de archivos: NTFS, ext4, FAT32.

  • Protección y comunicaciones.

  • Interfaces:

    • Línea de comandos (CLI).

    • Interfaz gráfica (GUI).

4. Sistemas Operativos Actuales

  • Clasificación:

    • Por usuarios: Monousuario vs. multiusuario.

    • Por licencia: Libre (Linux) vs. propietario (Windows/macOS).

  • Distribuciones Linux:

    • Ubuntu, Debian, Fedora.

5. Instalación y Configuración de SO

  • Requisitos:

    • Compatibilidad hardware (CPU, RAM, espacio en disco).

  • Fases:

    1. Configurar BIOS/UEFI para arranque.

    2. Particionado (MBR o GPT) y formateo (NTFS/ext4).

    3. Instalación del SO y controladores.

    4. Configuración post-instalación.

  • Tipos de instalación:

    • Atendida, desatendida, en red, clonación.

6. Replicación de Particiones y Discos

  • Herramientas:

    • Clonezilla, Acronis True Image.

  • Particiones:

    • Primarias, extendidas, lógicas.

  • Imágenes de disco:

    • Usadas para restauración rápida.

7. Actualización del Sistema Operativo

  • Fuentes de actualización:

    • Automática (Windows Update), manual (repositorios Linux).

  • Componentes críticos:

    • Parches de seguridad, controladores.

  • Documentación:

    • Registrar versiones y cambios realizados.

Preguntas Frecuentes en Exámenes

  1. Describe las fases de instalación de un SO.

    • Respuesta: Configuración BIOS, particionado, formateo, instalación, post-configuración.

  2. Diferencia entre instalación atendida y desatendida.

    • Respuesta: La atendida requiere interacción; la desatendida usa archivos de configuración automáticos.

  3. Tipos de memoria en un ordenador.

    • Respuesta: RAM (volátil), ROM (no volátil), caché (alta velocidad).

  4. Ejemplos de sistemas de archivos.

    • Respuesta: NTFS (Windows), ext4 (Linux), FAT32 (universal).

  5. Importancia de actualizar el SO.

    • Respuesta: Mejora seguridad, rendimiento y compatibilidad.

Consejos para el Examen

  • Práctica:

    • Usa VirtualBox para simulaciones.

    • Experimenta con herramientas como GParted.

  • Teoría:

    • Enfócate en definiciones (kernel, procesos, particiones).

  • Documentación:

    • Registra pasos en ejercicios prácticos (requisito profesional).
