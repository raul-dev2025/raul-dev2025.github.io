Cuestionario de Excel – Parte 3

Nombre: ___________________________ Fecha: ____________________________

—

1. ¿Cuál es la función de Excel que se utiliza para sumar un rango de celdas?

a) CONTAR  b) SUMA  c) SI  d) PROMEDIO Respuesta: b

—

2. ¿Qué símbolo se utiliza para comenzar una fórmula en Excel?

a) *  b) -  c) +  d) = Respuesta: d

—

3. ¿Cuál es la función que devuelve el número de celdas que contienen números en un rango?

a) CONTAR.VALORES  b) CONTAR.NUM  c) CONTAR  d) CONTAR.SI Respuesta: c

—

4. ¿Qué función se usa para calcular el promedio de un conjunto de números?

a) MEDIANA  b) PROMEDIO  c) SUMA  d) MODA Respuesta: b

—

5. ¿Qué hace la función SI en Excel?

a) Cuenta celdas  b) Evalúa una condición  c) Suma valores  d) Ordena datos Respuesta: b

—

6. ¿Qué función se utiliza para buscar un valor en la primera columna de una tabla y devolver un valor en la misma fila?

a) INDICE  b) BUSCARV  c) BUSCAR  d) COINCIDIR Respuesta: b

—

7. ¿Cuál es el atajo de teclado para copiar en Excel?

a) Ctrl + V  b) Ctrl + C  c) Ctrl + Z  d) Ctrl + X Respuesta: b

—

8. ¿Qué tipo de gráfico es más adecuado para mostrar proporciones?

a) Gráfico de líneas  b) Gráfico de dispersión  c) Gráfico circular  d) Gráfico de barras Respuesta: c

—

9. ¿Qué función devuelve el valor más alto en un rango?

a) PROMEDIO  b) SUMA  c) MAX  d) MIN Respuesta: c

—

10. ¿Qué hace el comando ‘Ordenar de A a Z’ en Excel?

a) Ordena por fecha descendente  b) Ordena de mayor a menor  c) Ordena por color de celda  d) Ordena alfabéticamente ascendente Respuesta: d

—

11. ¿Qué función se usa para contar celdas que cumplen una condición?

a) CONTAR.SI  b) CONTAR  c) SUMAR.SI  d) SI Respuesta: a

—

12. ¿Qué hace la función CONCATENAR?

a) Une textos  b) Cuenta caracteres  c) Suma números  d) Convierte texto en número Respuesta: a

—

13. ¿Qué significa el error #¡DIV/0! en Excel?

a) División por cero  b) Valor no disponible  c) Referencia no válida  d) Error de nombre Respuesta: a

—

14. ¿Cuál es el atajo para guardar un archivo en Excel?

a) Ctrl + G  b) Ctrl + S  c) Ctrl + A  d) Ctrl + P Respuesta: a

—

15. ¿Qué función se usa para redondear un número hacia abajo?

a) REDONDEAR  b) REDONDEAR.MAS  c) TRUNCAR  d) REDONDEAR.MENOS Respuesta: d
