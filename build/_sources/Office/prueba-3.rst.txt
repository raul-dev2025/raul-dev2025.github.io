Prueba 3
===========

Examen Teórico de Microsoft Word - 20 Preguntas
--------------------------------------------------

1. **¿Qué combinación de teclas abre el cuadro de diálogo "Fuente" en Word con teclado español?**
a) Ctrl + F
b) Ctrl + D
c) Ctrl + T
d) Ctrl + U

2. **¿Dónde se encuentra la opción para crear un nuevo estilo de párrafo?**
a) Pestaña "Inicio" → grupo "Estilos"
b) Pestaña "Diseño" → grupo "Formato"
c) Pestaña "Insertar" → grupo "Opciones"
d) Pestaña "Vista" → grupo "Estilos"

3. **¿Qué tecla rápida se usa para copiar formato en Word con teclado español?**
a) Ctrl + C
b) Ctrl + Mayús + C
c) Ctrl + Alt + C
d) Ctrl + V

4. **¿Cómo se inserta una marca de agua en un documento?**
a) Pestaña "Diseño" → "Marca de agua"
b) Pestaña "Insertar" → "Imagen" → "Marca"
c) Pestaña "Formato" → "Fondo"
d) Pestaña "Archivo" → "Propiedades"

5. **¿Qué opción permite convertir texto en columnas periodísticas?**
a) Diseño de página → Columnas
b) Formato → Columnas
c) Insertar → Columnas
d) Vista → Columnas

6. **¿Dónde se configura el interlineado a 1.5 líneas?**
a) Inicio → Párrafo → Interlineado
b) Diseño → Espaciado
c) Vista → Zoom
d) Insertar → Espaciado

7. **¿Qué combinación de teclas inserta un salto de línea sin cambiar párrafo?**
a) Mayús + Enter
b) Ctrl + Enter
c) Alt + Enter
d) Tab + Enter

8. **¿Cómo se activa la vista de esquema en Word?**
a) Vista → Esquema
b) Diseño → Esquema
c) Insertar → Esquema
d) Revisar → Esquema

9. **¿Qué opción permite crear un formulario editable en Word?**
a) Desarrollador → Controles de contenido
b) Insertar → Formulario
c) Diseño → Plantillas
d) Archivo → Nuevo formulario

10. **¿Cómo se inserta un símbolo matemático que no está en el teclado?**
a) Insertar → Símbolo
b) Diseño → Ecuación
c) Revisar → Matemáticas
d) Vista → Símbolos

11. **¿Qué tecla rápida selecciona todo el documento en español?**
a) Ctrl + A
b) Ctrl + E
c) Ctrl + T
d) Ctrl + Mayús + A

12. **¿Dónde se encuentra la opción para comparar dos documentos?**
a) Revisar → Comparar
b) Vista → Comparación
c) Archivo → Comparar
d) Diseño → Diferencias

13. **¿Qué combinación aplica subrayado en teclado español?**
a) Ctrl + S
b) Ctrl + U
c) Ctrl + Sub
d) Ctrl + Mayús + U

14. **¿Cómo se crea un encabezado diferente en la primera página?**
a) Diseño → Primera página diferente
b) Insertar → Encabezado → Opciones
c) Formato → Encabezado especial
d) Vista → Encabezado único

15. **¿Qué opción permite convertir texto a tabla?**
a) Insertar → Tabla → Convertir texto
b) Diseño → Convertir → Tabla
c) Inicio → Párrafo → Tabla
d) Vista → Conversión

16. **¿Dónde se ajusta el sangrado de primera línea?**
a) Inicio → Párrafo → Sangría
b) Diseño → Márgenes
c) Insertar → Sangría
d) Vista → Formato

17. **¿Qué tecla rápida guarda como en Word español?**
a) F12
b) Ctrl + G
c) Ctrl + Mayús + G
d) Alt + G

18. **¿Cómo se inserta una cita bibliográfica?**
a) Referencias → Insertar cita
b) Insertar → Referencia
c) Revisar → Bibliografía
d) Diseño → Citas

19. **¿Qué opción permite crear un hipervínculo a un marcador?**
a) Insertar → Hipervínculo → Marcador
b) Diseño → Enlace
c) Formato → Hipervínculo
d) Vista → Marcadores

20. **¿Cómo se activa el modo de pantalla completa?**
a) Vista → Pantalla completa
b) F11
c) Alt + V + U
d) Todas las anteriores

**Respuestas correctas:**
1. b), 2. a), 3. b), 4. a), 5. a), 6. a), 7. a), 8. a), 9. a), 10. a), 11. a), 12. a), 13. b), 14. b), 15. a), 16. a), 17. a), 18. a), 19. a), 20. d)
