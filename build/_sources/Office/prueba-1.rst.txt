=============

Examen Teórico de Microsoft Word - 20 Preguntas
--------------------------------------------------

1. **¿Qué pestaña de Word contiene las opciones para cambiar el tamaño y tipo de fuente?**  
   a) Insertar  
   b) Diseño  
   c) Inicio  
   d) Revisar  

2. **¿Cuál es la combinación de teclas para guardar un documento en Word?**  
   a) Ctrl + G  
   b) Ctrl + S  
   c) Ctrl + P  
   d) Ctrl + A  

3. **¿Qué opción permite alinear un párrafo a ambos márgenes (izquierdo y derecho)?**  
   a) Alinear a la izquierda  
   b) Centrar  
   c) Justificar  
   d) Alinear a la derecha  

4. **¿Dónde se encuentra la herramienta para insertar tablas en Word?**  
   a) Pestaña "Insertar" → "Tabla"  
   b) Pestaña "Diseño" → "Tabla"  
   c) Pestaña "Inicio" → "Tabla"  
   d) Pestaña "Vista" → "Tabla"  

5. **¿Qué función tiene la tecla F7 en Word?**  
   a) Abrir el cuadro de buscar  
   b) Revisar ortografía y gramática  
   c) Insertar un salto de página  
   d) Activar el control de cambios  

6. **¿Qué formato de archivo conserva el formato pero no permite edición fácil?**  
   a) .docx  
   b) .txt  
   c) .pdf  
   d) .rtf  

7. **¿Cómo se llama la herramienta que muestra los cambios realizados por otros usuarios?**  
   a) Comentarios  
   b) Control de cambios  
   c) Seguimiento de revisiones  
   d) Comparar documentos  

8. **¿Qué opción permite crear una lista numerada automáticamente?**  
   a) Viñetas  
   b) Numeración  
   c) Lista multinivel  
   d) Sangría  

9. **¿Dónde se configura el tamaño y la orientación del papel en Word?**  
   a) Pestaña "Diseño" → "Configurar página"  
   b) Pestaña "Inicio" → "Párrafo"  
   c) Pestaña "Insertar" → "Encabezado"  
   d) Pestaña "Revisar" → "Ortografía"  

10. **¿Qué comando se usa para pegar texto manteniendo el formato original?**  
    a) Pegar  
    b) Pegar especial  
    c) Pegar como texto  
    d) Pegar formato  

11. **¿Qué herramienta permite combinar dos celdas en una tabla?**  
    a) Dividir celdas  
    b) Combinar celdas  
    c) Insertar fila  
    d) Eliminar tabla  

12. **¿Cuál es la función de la tecla "Ctrl + Z"?**  
    a) Copiar  
    b) Pegar  
    c) Deshacer  
    d) Rehacer  

13. **¿Qué opción permite insertar un salto de página manual?**  
    a) Ctrl + Enter  
    b) Ctrl + Espacio  
    c) Ctrl + P  
    d) Ctrl + B  

14. **¿Dónde se agrega un encabezado o pie de página en Word?**  
    a) Pestaña "Insertar" → "Encabezado y pie de página"  
    b) Pestaña "Diseño" → "Márgenes"  
    c) Pestaña "Revisar" → "Comentarios"  
    d) Pestaña "Vista" → "Inmersiva"  

15. **¿Qué vista de documento muestra cómo se verá al imprimirse?**  
    a) Borrador  
    b) Diseño de impresión  
    c) Diseño web

16. **¿Qué herramienta permite buscar sinónimos de una palabra?**

    a) Ctrl + K  
    b) Ctrl + H

18. **¿Qué opción permite ajustar el interlineado de un párrafo?**

19. **¿Qué comando se usa para imprimir un documento?**
