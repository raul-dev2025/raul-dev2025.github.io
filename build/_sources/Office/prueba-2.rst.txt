Prueba 2
==========

Examen Teórico de Microsoft Word - 20 Preguntas
-------------------------------------------------------------------------

1. **¿Qué combinación de teclas se usa para poner texto en negrita en Word con teclado en español?**
a) Ctrl + B
b) Ctrl + N
c) Alt + N
d) Ctrl + K

2. **¿Dónde se encuentra la opción para cambiar la orientación de página a horizontal?**
a) Pestaña "Diseño" → "Orientación"
b) Pestaña "Inicio" → "Párrafo"
c) Pestaña "Vista" → "Zoom"
d) Pestaña "Insertar" → "Página"

3. **¿Qué tecla rápida se usa en español para deshacer la última acción?**
a) Ctrl + Z
b) Ctrl + Y
c) Ctrl + D
d) Ctrl + U

4. **¿Cómo se inserta una tabla de contenidos automática?**
a) Pestaña "Referencias" → "Tabla de contenido"
b) Pestaña "Insertar" → "Tabla"
c) Pestaña "Diseño" → "Índice"
d) Pestaña "Revisar" → "Resumen"

5. **¿Qué opción permite ver los caracteres no imprimibles como espacios y saltos de párrafo?**
a) ¶ (Mostrar todo)
b) Ctrl + *
c) Alt + F10
d) Ambas a y b son correctas

6. **¿Dónde se activa el corrector ortográfico en español?**
a) Pestaña "Revisar" → "Ortografía"
b) Pestaña "Inicio" → "Fuente"
c) Pestaña "Archivo" → "Opciones"
d) Pestaña "Vista" → "Inmersión"

7. **¿Qué combinación de teclas centra un párrafo en Word con teclado español?**
a) Ctrl + T
b) Ctrl + C
c) Ctrl + J
d) Ctrl + E

8. **¿Cómo se inserta un salto de sección en Word?**
a) Pestaña "Diseño" → "Saltos"
b) Pestaña "Insertar" → "Salto"
c) Pestaña "Formato" → "Sección"
d) Pestaña "Disposición" → "Pausas"

9. **¿Qué opción permite proteger un documento con contraseña?**
a) Archivo → Información → Proteger documento
b) Revisar → Restringir edición
c) Diseño → Proteger
d) Ambas a y b son correctas

10. **¿Cómo se cambia el idioma de corrección para un texto seleccionado?**
a) Revisar → Idioma → Establecer idioma de corrección
b) Inicio → Fuente → Idioma
c) Archivo → Opciones → Idioma
d) Vista → Idioma

11. **¿Qué tecla rápida abre el cuadro de buscar y reemplazar en español?**
a) Ctrl + B
b) Ctrl + L
c) Ctrl + F
d) Ctrl + R

12. **¿Dónde se encuentra la opción para numerar páginas?**
a) Insertar → Número de página
b) Diseño → Encabezados
c) Vista → Numeración
d) Referencias → Marcadores

13. **¿Qué combinación aplica cursiva en teclado español?**
a) Ctrl + K
b) Ctrl + I
c) Ctrl + C
d) Ctrl + S

14. **¿Cómo se crea un índice automático?**
a) Referencias → Insertar índice
b) Insertar → Tabla de contenido
c) Revisar → Resumen
d) Diseño → Índice automático

15. **¿Qué opción permite guardar como PDF?**
a) Archivo → Exportar → PDF
b) Archivo → Guardar como → PDF
c) Archivo → Imprimir → PDF
d) Todas las anteriores

16. **¿Dónde se ajustan los márgenes del documento?**
a) Diseño → Márgenes
b) Archivo → Configurar página
c) Vista → Diseño de impresión
d) Insertar → Márgenes

17. **¿Qué tecla rápida justifica un párrafo en español?**
a) Ctrl + J
b) Ctrl + Q
c) Ctrl + F
d) Ctrl + G

18. **¿Cómo se inserta una nota al pie?**
a) Referencias → Insertar nota al pie
b) Insertar → Comentario
c) Revisar → Anotación
d) Vista → Notas

19. **¿Qué opción muestra la regla en Word?**
a) Vista → Mostrar → Regla
b) Archivo → Opciones → Regla
c) Diseño → Reglas
d) Insertar → Regla

20. **¿Cómo se activa el control de cambios?**
a) Revisar → Control de cambios
b) Vista → Seguimiento
c) Archivo → Control de versión
d) Diseño → Revisiones

**Respuestas correctas:**
1. b), 2. a), 3. a), 4. a), 5. d), 6. a), 7. d), 8. a), 9. d), 10. a), 11. b), 12. a), 13. b), 14. a), 15. d), 16. a), 17. a), 18. a), 19. a), 20. a)
